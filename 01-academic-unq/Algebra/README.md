## 📘 Descripción / Description
**Español:**  
LogicInterpreter es un motor en Python para evaluar expresiones lógicas y matemáticas.  
Convierte símbolos de lógica proposicional (∧, ∨, ~, →, ↔) y comparaciones aritméticas en operadores de Python, permitiendo analizar valores de verdad de manera interactiva.  
El proyecto está en desarrollo y evolucionará hacia una **herramienta múltiple (multi-tool)** que combine lógica, matemáticas y razonamiento.

**English:**  
LogicInterpreter is a Python engine to evaluate logical and mathematical expressions.  
It translates propositional logic symbols (∧, ∨, ~, →, ↔) and arithmetic comparisons into Python operators, enabling interactive truth-value analysis.  
The project is under development and will evolve into a **multi-tool** combining logic, mathematics, and reasoning.

---

## ✅ Funcionalidades actuales / Current Features
- ✅ Detección de variables proposicionales (`p`, `q`, etc.)  
- ✅ Conectivos lógicos soportados: ∧, ∨, ~, ¬, →, ↔  
- ✅ Comparaciones aritméticas (`=`, `>`, `<`) y multiplicación (`x`)  
- ✅ Asignación interactiva de valores de verdad  
- ✅ Evaluación con `eval` en Python  
- ✅ Filtro básico para evitar entradas no proposicionales
